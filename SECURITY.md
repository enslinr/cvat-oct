# Security Policy

This is a research fork of [CVAT](https://github.com/cvat-ai/cvat), maintained as part of an
MSc dissertation. It is provided as-is for academic and research purposes and is not
production-hardened.

## Reporting an issue

- For security issues in **upstream CVAT**, please report to the CVAT.ai team via their
  [security policy](https://github.com/cvat-ai/cvat/security).
- For issues specific to **this fork's customizations** (the SAM2-OCT interactor,
  masks-to-polygons action, or setup scripts), please open a
  [GitHub issue](https://github.com/enslinr/cvat-oct/issues).

Please do not deploy this fork in a security-sensitive or production environment without
your own review.
